Une fonction qui récupère le nom, le prénom et le numéro d\textquotesingle{}étudiant, puis elle récupère le hash du vote correspondant. Enfin, elle vérifie si le hash est présent dans le fichier des résultats des votes. Si c\textquotesingle{}est le cas, elle affiche les détails du vote, sinon elle affiche un message d\textquotesingle{}erreur.
