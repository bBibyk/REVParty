Structure de données d\textquotesingle{}une colonne. 

Structure de données représentant une colonne dans un \mbox{\hyperlink{structDataFrame}{Data\+Frame}}.

Cette structure permet de stocker les informations d\textquotesingle{}une colonne d\textquotesingle{}un \mbox{\hyperlink{structDataFrame}{Data\+Frame}}. Elle contient le nom de la colonne, son type de données et un tableau de données.
