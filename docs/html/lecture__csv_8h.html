Fonction de recherche d'une valeur dans une colonne d'un DataFrame à partir de son nom et de sa valeur (int, double ou string uniquement)

Fonction de recherche d'une valeur dans une colonne d'un DataFrame.

Paramètres
[in]dfDataFrame*
[in]column_namechar*
[in]valuechar*
Renvoie
int

Cette fonction permet de verifier si une valeur est présente dans une colonne d'un DataFrame. Si elle est présente, la fonction retourne l'indice de la ligne, -1 sinon.

Note
Si la colonne n'existe pas, la fonction retourne false.
Paramètres
[in]dfPointeur vers le DataFrame dans lequel effectuer la recherche.
[in]column_nameNom de la colonne dans laquelle rechercher.
[in]valueValeur à rechercher (int, double ou string).
Renvoie
true si la valeur est trouvée, false sinon.

Cette fonction recherche une valeur donnée dans une colonne d'un DataFrame. Si la colonne ou la valeur n'est pas trouvée, elle renvoie false. Sinon, elle renvoie true.
