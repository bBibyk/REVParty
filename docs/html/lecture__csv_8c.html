enum DataType

Enumération des types de données.

Cette énumération permet de définir les types de données possibles pour une colonne (INT, DOUBLE, TIMESTAMP, STRING).

Valeurs énumérées
INT 

Entier.

DOUBLE 

Nombre à virgule flottante.

TIMESTAMP 

Horodatage (timestamp).

STRING 

Chaîne de caractères.
